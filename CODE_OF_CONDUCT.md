# Code of Conduct

## Our standard

Launchboard is an open-source project and contributor community. Participation is expected to stay respectful, constructive, and professional.

Examples of behavior that help the project:

- being direct without being hostile
- assuming good intent while still giving clear technical feedback
- welcoming contributors with different backgrounds and experience levels
- focusing criticism on code, design, and product decisions rather than people
- documenting context so others can understand and build on the work

Examples of unacceptable behavior:

- harassment, discrimination, or hateful conduct
- personal attacks, insults, or intimidation
- doxxing or sharing private information without permission
- trolling, bad-faith disruption, or deliberately derailing discussion
- sexualized language or unwanted sexual attention

## Scope

This applies in GitHub issues, pull requests, discussions, documentation, and other project spaces.

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, or contributions that do not follow this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers through the contact method listed in [SECURITY.md](/Users/briantehsayy/Desktop/launchboard/SECURITY.md) or the repository owner contact listed on GitHub.

## Notes

This policy is intentionally simple: be rigorous, be respectful, and do not make the project harder for others to safely contribute to.
